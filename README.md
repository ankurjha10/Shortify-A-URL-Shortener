<h2>What this project does</h2>
<p>
Shortify takes any long URL and generates a short unique link. Opening that short link redirects the user back to the original URL. Each click is counted and stored, so you can also view basic usage stats and track link performance over time.
</p>

<h2>Why this project was built</h2>
<p>
This is a practice full stack project to understand how real products work end-to-end. The goal was not just shortening links, but learning how to connect frontend, backend, database, CORS, API calls, deployment, hosting and environment configs together in a clean pipeline.
</p>

<h2>Tech used</h2>
<ul>
  <li>React + Vite (TypeScript) for UI</li>
  <li>Tailwind + shadcn/ui for design + components</li>
  <li>Spring Boot (Java) for backend logic</li>
  <li>MySQL for storing links & analytics</li>
  <li>Vercel (Frontend Deploy)</li>
  <li>Railway (Backend Deploy)</li>
</ul>
